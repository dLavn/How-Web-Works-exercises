## **Part One: Solidify Terminology** - What is HTTP? Hyper Text Transfer Protocol (how data is sent to and from servers). - What is a URL? Uniform Resource Locator (address/site string to navigate to a website) - What is DNS? Domain Name System (converts URLs into IP Addresses) - What is a query string? A query string lets you format key-value pairs into a URL. - What are two HTTP verbs and how are they different? Get and Post. Get receives data from the server, and Post sends data to the server. - What is an HTTP request? Requests come from clients and go to servers - What is an HTTP response? Responses come from servers and go to clients - What is an HTTP header? Give a couple examples of request and response headers you have seen. Headers give more information about the HTTP request/response. Some HTTP request headers include Host, Accept, and User-Agent. Some HTTP response headers include Last-Modified and Set-Cookie. - What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser? When you type a URL into a browser, 1. Your browser uses DNS to convert the URL into an IP address 2. The browser makes an HTTP request to the IP address 3. The server sends an HTTP response to the client 4. The browser makes a DOM from that HTML, and finds other CSS or JavaScript associated with the HTML 5. The browser makes separate HTTP requests for those other resources and gets a separate HTTP response from the server for each individual request *********************************************** ## **Part Two: Practice Tools** 1.
